Format crates for the [`facet`](https://github.com/facet-rs/facet) ecosystem.

This repository hosts:

- `facet-format`
- `facet-dessert`
- `facet-json`
- `facet-json-classics`
- `facet-postcard`
- `facet-msgpack`
- `facet-toml`
- `facet-yaml`
- `facet-format-suite`
- `facet-asn1`
- `facet-csv`
- `facet-xdr`
- `facet-json-schema`
- `facet-typescript`
- `facet-python`
- `facet-value-format`

Core reflection crates such as `facet`, `facet-core`, `facet-reflect`, `facet-path`,
`facet-solver`, `facet-value`, and related macro crates remain in the main
[`facet`](https://github.com/facet-rs/facet) repository.
